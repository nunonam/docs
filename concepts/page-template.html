Single-line code block

A single-line code block is defined as below. For proper language formatting, providing the language type is necessary.

Copied
$mysql --host {hostname} --port {port number} -p --user {username}

Multi-line code block

Multi-line code blocks are defined with the same syntax as single-line code blocks.

Copied
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 83597
Server version: 5.7.9-Vitess


Copyright (c) 2000, 2020, Oracle and/or its affiliates. All rights reserved.


Oracle is a registered trademark of Oracle Corporation and/or its
affiliates. Other names may be trademarks of their respective
owners.


Type 'help;' or '\h' for help. Type '\c' to clear the current input statement.


mysql>

Split cmd+output code blocks

Code blocks can also be configured to have a split view of a command and corresponding output by delimiting the two pieces with a ------ string. In such a case, the copy button will only copy the command and not the output.

Copied
$mysql --host {hostname} --port {port number} -p --user {username}
------
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 83597
Server version: 5.7.9-Vitess


Copyright (c) 2000, 2020, Oracle and/or its affiliates. All rights reserved.


Oracle is a registered trademark of Oracle Corporation and/or its
affiliates. Other names may be trademarks of their respective
owners.


Type 'help;' or '\h' for help. Type '\c' to clear the current input statement.


mysql>

Theme sensitive images

Images benefit from a brief description which also serves as the ALT tag value (will be made independently configurable later). Theme sensitive images should be added in doubles; one with a _light suffix and with _dark. The image with the _light suffix should be the one that is linked within the markdown. The system will use the dark mode image appropriately.

Organization dashboard

Theme agnostic images

Images can also be added to be theme agnostic i.e. they will not change in anyway regardless of page's theme. These images don't have a special suffix format.

Organization Settings > Members
